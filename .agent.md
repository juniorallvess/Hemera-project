# Hemera Multi-Platform Agent Configuration

This file configures agent discovery for VS Code, GitHub Copilot, and other platforms.

## Platform Compatibility

| Platform | Status | Usage |
|----------|--------|-------|
| **Antigravity** | ✅ Native | `@agent-name` |
| **VS Code** | ✅ Via This Config | `@agent-name` or `/agent agent-name` |
| **GitHub Copilot** | ✅ Via This Config | `@agent-name` |
| **CLI** | 🔄 Coming Soon | `hemera --agent agent-name` |

## How It Works

1. **Discovery**: Each agent is loaded from `agents/*.md`
2. **Triggers**: Keywords that auto-suggest the agent
3. **Invocation**: Use `@agent-name` to call directly
4. **Fallback**: Keywords match partial agent names

## Adding New Agents

Add entry to `agents:` list with:
- `name`: kebab-case identifier
- `file`: relative path to agent markdown
- `triggers`: keywords that trigger auto-suggestion
